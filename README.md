# PETANAS (PENCATATAN NASKAH DINAS KEMAHASISWAAN POLBAN)
Web application for Kemahasiswaan Polban

Technology  stack:
1. Laravel v.8 as Backend or Server Side
   Link Documentation : https://laravel.com/docs/8.x/installation
2. React Js as Frontend or Client Side
   Link Documentation : https://reactjs.org/docs/getting-started.html
4. Tailwind CSS as Styling or Markup
   Link Documentation : https://tailwindcss.com/
5. Firebase Storage as Cloud Storage
   Link Documentation : https://firebase.google.com/?hl=id
