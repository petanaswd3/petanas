# PETANAS (PENCATATAN NASKAH DINAS KEMAHASISWAAN POLBAN)

## Web application for Kemahasiswaan Polban
___
### Technology  stack:
1. Laravel v.8 as Backend or Server Side
   
   [Go to documentation]( https://laravel.com/docs/8.x/installation)
2. React Js as Frontend or Client Side
   
   [Link documentation](https://reactjs.org/docs/getting-started.html)
3. Tailwind CSS as Styling or Markup
   
   [Link documentation](  https://tailwindcss.com/)
4. Firebase Storage as Cloud Storage
   
   [Link documentation](
   https://firebase.google.com/?hl=id)

5. MySQL

   [Link documentation](https://www.mysql.com/)

___
## Development Scripts

1. Laravel
+ Update composer environments
   `composer update`
+ Run on local server `php artisan serve`

2. React js
+ Instal node module environments
  `npm install`
+ Run on local server `npm run start`
+ Build project  `npm run build`

3. Tailwind CSS 
+ Optimizing for Production `npx tailwindcss -o build.css --minify`

**To run this application, environment required**
+ node v.14.*
+ php v.8.*
